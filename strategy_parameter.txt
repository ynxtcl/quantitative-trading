【策略A】趋势跟踪 TrendFollowingStrategy
  文件：strategies/trend_following/strategy.py
  
  入场逻辑（三个条件必须同时满足）：
    ① 价格突破过去20日最高点（唐奇安通道突破 = entry_period）
    ② 价格在60日均线上方（ma_filter_period=60，不逆势做多）
    ③ ADX > 25（趋势强度足够，排除弱趋势/震荡市）
  
  额外过滤：当 close < EMA20 时不新开仓（ema_filter_enabled=True）
  
  出场逻辑（满足其一即触发）：
    ① 通道跌破：价格跌破过去20日最低点（exit_period=20）
    ② 分阶段止盈：从10日高点回落>8% → 平掉一半仓位
  
  ATR动态仓位：close/atr < 12 时按比例缩减仓位
  
  置信度：confidence = min(ADX / 50, 1.0)
  
  完整参数表：

【策略B】均值回归 MeanReversionStrategy
  文件：strategies/mean_reversion/strategy.py
  
  入场逻辑（四个条件同时满足 + 两个辅助条件）：
    ① 价格触及布林带下轨：close < bb_lower（bb_period=20, bb_std=2.0）
    ② RSI < 30（rsi_period=14, rsi_oversold=30，超卖状态）
    ③ 布林带宽度在[0.05, 0.20]之间（防止压缩入场/极端波动）
    ④ 成交量确认：缩量<60%均量 OR 恐慌放量>200%均量
    ⑤ 趋势过滤：价格 > EMA50 × 0.92（不接飞刀，2026-07-01自0.97降至0.92）
  
  出场逻辑（满足其一即触发）：
    ① 价格触及布林带上轨 + RSI > 70（rsi_overbought=70）
    ② ATR止损：跌破 bb_lower - 1.5×ATR（stop_loss_atr=1.5）
  
  动态仓位：
    偏离度Z-score越大仓位越高：Z=2→confidence=0.67, Z=3→1.0
    市场状态自适应（默认关闭）：普跌(mmr<-3%)→缩至30%，严重(mmr<-8%)→缩至15%

三、组合引擎 PortfolioEngine
=============================
  文件：portfolio/engine.py + portfolio/combiner.py

  每日执行流程：
    1. 前置计算：全市场中位数回报率(market_median_return)
    2. 运行择时策略：TF(数据→信号) + MR(注入market_median_return→信号)
    3. 因子选股：每月最后交易日打分配置再平衡信号
    4. 信号融合：combiner.combine() → 净权重求和
    5. 止损检查：固定止损(8%) + 移动止损(15%)
    6. 风控过滤：RiskManager.filter_signals() → 八重规则
    7. 批量执行信号 → 记录每日净值

  信号融合规则（净权重求和）：
    同一股票多策略冲突解决：
      TF买入 weight=0.40 + MR卖出 weight=0.15 = 净买入 0.25
    净权重 > 0 → 买入 | < 0 → 卖出 | = 0 → 无信号

  策略当前权重：
    趋势跟踪(TF)  : 35%
    均值回归(MR)  : 25%
    因子选股(FS)  : 40%


四、Walk-Forward交叉验证系统
=============================
  文件：backtest/walk_forward.py

  窗口设计：3年 = 前2年训练 + 后1年测试
  每年滑动推进 → 5年数据产生3轮验证

  验证轮次：
    第1轮：训练 2020-01~2021-12 → 测试 2022-01~2022-12
    第2轮：训练 2021-01~2022-12 → 测试 2023-01~2023-12
    第3轮：训练 2022-01~2023-12 → 测试 2024-01~2024-12

  OOS比率判断标准（测试夏普/训练夏普）：
    > 0.7   → ✅ 稳健，可进入组合/实盘
    0.3~0.7 → ⚠️ 中度过拟合，简化参数
    < 0.3   → ❌ 严重过拟合，重新设计

  参数：
    window_years = 3（总窗口3年）
    train_ratio  = 2/3（前2/3训练）
    step_years   = 1（每次推进1年）
